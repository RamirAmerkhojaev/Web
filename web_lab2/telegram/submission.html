Welcome to the official Telegram web-client.

About

The Telegram web-client is a free, fast and
secure way to enjoy most of Telegram's
features right in your web-browser.

It is always synced with the Telegram app on
your mobile device, which makes it a perfect
tool for messaging and file sharing.

Our source code is open, so everyone can
make a contribution.
